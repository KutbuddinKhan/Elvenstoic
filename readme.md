# 🌌 Elevenstoic — Cinematic Brand Website

Welcome to the Elevenstoic website — a high-converting, visually cinematic marketing site built to showcase and sell a complete Instagram content growth system. This site is optimized for performance, conversion, and professional design.

## 🔧 Built With

- **Next.js (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **GSAP (ScrollTrigger)**
- **Lucide Icons**
- **React Masonry CSS**
- **Wistia Video Embed**
- **Responsive Design (Mobile-first)**

---

## 📁 Project Structure

```bash
.
├── components/
│   ├── layout/          # Header, Footer, Logo
│   ├── home/            # All homepage sections (Hero, Reviews, FAQ, etc.)
│   ├── legal/           # Layout for legal pages
│   └── ui/              # Button, container, utilities
├── app/               # Page entry points (App Router if using latest Next.js)
├── public/              # Static assets (images, avatars, videos)
│   ├── avatars/
│   ├── fan-comments/
│   ├── instructor/
│   └── logo/
├── styles/              # Global styles (Tailwind)
├── lib/                 # Utility functions (e.g., cn)
└── app/                 # Route-based folder structure (for App Router)
````

## 📦 Features

* 🔹 **Cinematic Hero Section** with Wistia video and glowing typography
* 🔹 **Masonry Fan Wall** with lightbox and animations
* 🔹 **Animated Sections** using GSAP & Framer Motion
* 🔹 **Custom Gradient Spotlights** for visual storytelling
* 🔹 **Legal Pages** — Terms, Privacy, Refund, Imprint
* 🔹 **Responsive Navigation** with hover effects and gradient CTA
* 🔹 **Dark mode aesthetic** using slate and blue tones
* 🔹 **Conversion-focused CTA buttons** with subtle glow and animations
* 🔹 **Testimonial Section** with real Instagram stats and avatar-based trust

## 🚀 Getting Started

1. **Clone the repo**

```bash
git clone https://github.com/yourusername/elevenstoic-website.git
cd elevenstoic-website
```

2. **Install dependencies**

```bash
pnpm install
```

3. **Run locally**

```bash
pnpm run dev
```

4. **Visit**

```
http://localhost:3000
```
